Реализация структуры данных b-tree.
В ходе этой работы была изучена новая структура данных – B-дерево, основные методы (операции добавления, удаления узлов, а также поиск узла в дереве) которой были реализованы программно на языке C#. Кроме того, было проведено тестирование реализации данной структуры с помощью инструментов платформы VisualStudio, таких как UnitTestProject. После этого было проведено нагрузочное тестирование большим количеством данных для выяснения оптимального размера страницы.
